리눅스

리눅스는 디렉터리와 파일의 구분이 어렵다.

절대경로   상대경로
/사용        /미사용

리눅스의 관리자 계정 : root (super user 줄여서 su)
최초 root 계정의 비밀번호는 세팅해야한다.

sudo passwd root : 비밀번호 세팅

su - : root로 접속( putty접속 시 root접속이 막혀있음 - 최초 접속 시에는 사용자 계정(guru)으로 접속 후 변경)
logout : root 로그아웃


디렉터리 구조
/  -> home -> guru 
(루트)            (사용자)
                    = home directory
	        = ~

명령어
pwd : 현재 디렉터리 확인
cd : 디렉터리 변경 (cd . cd .. cd ~ cd guru 등)
clear : 화면 정리
crtl + c : 작업취소
exit : 종료
ls : 디렉터리 목록 보기
ls -a : 숨겨진 디렉터리까지 모두 보기
-l : 자세히보기
cp : 복사하기
vi : 탐색기
rm (파일명) : 파일 삭제
rm -i (파일명) : 파일삭제 여부 확인
alias rm='rm -i' : rm으로 등록
cat (파일명) : 파일 내용 보기 
alias : 등록된 alias 확인하기

 vi (파일명) 
 vi 탐색기로 들어가면 명령모드로 시작.
 명령모드에서 입력모드로 가려면 a, i, o 입력 필요.
                                         (a : 커서 뒤에, i : 커서 앞에, o : 커서 아래)
 입력모드를 빠져나오려면 esc 사용 -> 다시 명령모드로 나옴.
 
 명령모드에서 콜론모드로 넘어갈 때 콜론(:) 사용
 콜론모드에서 명령모드로 갈때는 아무키나 입력

콜론모드(lastline) - :누르면 시작
wq : 저장닫기
w (파일명) : 파일명에 저장하기
q : 그냥나가기
q! : 저장없이 나오기

명령모드
$ : 라인의 맨 뒤
^ : 라인의 맨 앞
G : 마지막
u : 취소
